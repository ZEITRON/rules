Правила игры

Существует только три игровых класса [волшебник] [кузнец] [военный]

1.
Нельзя гриферить чужие территории/При несоблюдении правил [Наказание: - 50% ресурсов]

2.
Человек играющий за класс Военного может использовать только 3 артефактa
[вечный стейк] [зонтики] [кошачьи тапочки]

3.
Инструмент из мода Tinker Construct может создать только кузнец, Перековка и вставление кристалла занимается кузнец, Кузнец может ставить заклинание только свои изготовленные предметы

4.
Кузнец может ставить заклинания 1 и 2 уровня но по достижению 100 уровня Может ставить заклинания до 5 уровня

5.
Кузнец может использовать артефакты только те которые ему разрешили

6.
У Мага нет никаких ограничений по использованию предметов

7.
Военный может использовать снайперку только с достижением 60 уровня

8.
Ограничение по использованию ванильных предметов нету

9.
Любой человек который нашёл и убил босса и с него получил редкий Предмет сможет его использовать

10.
На сервере ничего просто так нельзя давать всё должно быть на бартерной основе

11.
За Читы бан, За оскорбление кого-нибудь расстрел потом бан

12.
Магический Кристалл телепортации может использовать только волшебник

13.
Волшебник может использовать высшие заклинания только с 120 уровня

14.
Зачаровывать предметы может только волшебник
